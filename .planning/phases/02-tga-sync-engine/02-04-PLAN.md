    Create `apps/api/src/tga/tga-import.service.ts` as a @Injectable() NestJS service:

    **Method: `importQualification(rtoId: string, qualCode: string): Promise<ImportResult>`**

    Implementation steps (all in one DB transaction where possible):

    1. Fetch qualification detail from TGA: `await this.tgaApiClient.getQualificationDetail(qualCode)`
       - If null/not found: throw NotFoundException('Qualification not found on TGA')

    2. Upsert the Qualification record:
       ```typescript
       const qual = await prisma.qualification.upsert({
         where: { code: qualCode },
         create: {
           code: tgaQual.code,
           title: tgaQual.title,
           status: tgaQual.status,
           superseded_by: tgaQual.supersededBy ?? null,
           training_package: tgaQual.trainingPackage?.code ?? null,
           tga_content_hash: this.tgaSyncService.computeQualificationHash(tgaQual),
           last_synced_at: new Date(),
         },
         update: {
           title: tgaQual.title,
           status: tgaQual.status,
           superseded_by: tgaQual.supersededBy ?? null,
           training_package: tgaQual.trainingPackage?.code ?? null,
           tga_content_hash: this.tgaSyncService.computeQualificationHash(tgaQual),
           last_synced_at: new Date(),
         },
       });
       ```

    3. Extract all unit codes from TGA qualification:
       ```typescript
       const unitCodes = tgaQual.unitGroups.flatMap(g => g.units.map(u => u.code));
       ```

    4. For each unit code (process sequentially, add 50ms delay between TGA calls):
       a. Fetch unit detail: `await this.tgaApiClient.getUnitDetail(unitCode)`
       b. If null: log warning, skip this unit
       c. Upsert Unit record (same upsert pattern as qualification above)
       d. Replace UnitElements for this unit: delete old `unit_elements` rows, insert fresh from TGA
          ```typescript
          await prisma.unitElement.deleteMany({ where: { unit_id: unit.id } });
          for (const el of tgaUnit.elements) {
            const element = await prisma.unitElement.create({
              data: { unit_id: unit.id, element_num: el.num, title: el.title },
            });
            await prisma.performanceCriterion.createMany({
              data: el.performanceCriteria.map(pc => ({
                element_id: element.id, pc_num: pc.num, text: pc.text,
              })),
            });
          }
          ```
       e. Upsert QualificationUnit join:
          ```typescript
          await prisma.qualificationUnit.upsert({
            where: { qualification_id_unit_id: { qualification_id: qual.id, unit_id: unit.id } },
            create: { qualification_id: qual.id, unit_id: unit.id },
            update: {},
          });
          ```

    5. Upsert RtoQualification:
       - Check if RTO already has this qualification linked (by rto_id + qualification_id + deleted_at null)
       - If yes: update is_active = true (idempotent re-import)
       - If no: create new RtoQualification with rto_id and qualification_id
       - Also soft-delete any stub RtoQualification rows for this RTO where qualification_id IS NULL:
         ```typescript
         await prisma.rtoQualification.updateMany({
           where: { rto_id: rtoId, qualification_id: null, deleted_at: null },
           data: { deleted_at: new Date() },
         });
         ```

    6. Return: `{ qualificationId: qual.id, code: qual.code, title: qual.title, unitCount: unitCodes.length }`

    Add `TgaImportService` to tga.module.ts providers (inject TgaApiClient and TgaSyncService).

    **1. Add TGA API call helpers to apps/web/src/lib/api.ts** (or wherever fetch helpers live):
    ```typescript
    export async function searchTgaQualifications(query: string, token: string) {
      const res = await fetch(`/api/v1/tga/qualifications/search?q=${encodeURIComponent(query)}`, {
        headers: { Authorization: `Bearer ${token}` },
      });
      if (!res.ok) throw new Error('TGA search failed');
      return res.json();
    }

    export async function importTgaQualification(rtoId: string, qualificationCode: string, token: string) {
      const res = await fetch(`/api/v1/tga/rtos/${rtoId}/qualifications/import`, {
        method: 'POST',
        headers: { Authorization: `Bearer ${token}`, 'Content-Type': 'application/json' },
        body: JSON.stringify({ qualificationCode }),
      });
      if (!res.ok) {
        const err = await res.json().catch(() => ({}));
        throw new Error(err.message ?? 'Import failed');
      }
      return res.json();
    }

    export async function getRtoQualifications(rtoId: string, token: string) {
      const res = await fetch(`/api/v1/tga/rtos/${rtoId}/qualifications`, {
        headers: { Authorization: `Bearer ${token}` },
      });
      if (!res.ok) throw new Error('Failed to load qualifications');
      return res.json();
    }
    ```
