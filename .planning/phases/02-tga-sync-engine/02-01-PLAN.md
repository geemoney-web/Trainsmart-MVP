       - id (UUID PK), qualification_id @db.Uuid FK to Qualification, unit_id @db.Uuid FK to Unit
       - created_at DateTime @default(now()) @db.Timestamptz
       - @@unique([qualification_id, unit_id])
       - @@index([qualification_id]), @@index([unit_id])

       - id (UUID PK), unit_id @db.Uuid FK to Unit
       - element_num String ("1", "2", "3")
       - title String
       - created_at, updated_at
       - Relations: performance_criteria PerformanceCriterion[]
       - @@index([unit_id])
       - @@map("unit_elements")

    5. **PerformanceCriterion**:
       - id (UUID PK), element_id @db.Uuid FK to UnitElement
       - pc_num String ("1.1", "1.2")
       - text String
       - created_at, updated_at
       - @@index([element_id])

       - id (UUID PK), qualification_id @db.Uuid FK to Qualification
       - snapshot_data Json (full TGA response at time of change)
       - change_fields String[] (which fields changed)
       - snapshotted_at DateTime @default(now()) @db.Timestamptz
       - @@index([qualification_id]), @@index([snapshotted_at])
       - @@map("qualification_snapshots")

    7. **UnitSnapshot** (immutable, no soft delete):
       - id (UUID PK), unit_id @db.Uuid FK to Unit
       - snapshot_data Json (full TGA response including elements + PCs)
       - change_fields String[]
       - snapshotted_at DateTime @default(now()) @db.Timestamptz
       - @@index([unit_id]), @@index([snapshotted_at])
       - @@map("unit_snapshots")

    8. **SyncLog**:
       - id (UUID PK), triggered_by String ("nightly_cron" | "manual:{userId}")
       - status String @default("running") (running | completed | failed)
       - quals_checked Int @default(0), quals_changed Int @default(0)
       - units_checked Int @default(0), units_changed Int @default(0)
       - alerts_created Int @default(0)
       - error_message String?
       - started_at DateTime @default(now()) @db.Timestamptz
       - completed_at DateTime? @db.Timestamptz
       - @@index([status]), @@index([started_at])
       - @@map("sync_logs")

    - entity_type String? ("qualification" | "unit" | "trainer" | "tas_document")

    1. **apps/api/src/tga/dto/tga-qualification.dto.ts** — TypeScript interfaces for TGA API responses:
       ```typescript
       export interface TgaQualificationSearchResult {
         code: string;
         title: string;
         status: string; // "Current" | "Superseded" | "Deleted"
       }

       export interface TgaQualificationDetail {
         code: string;
         title: string;
         status: string;
         supersededBy: string | null;
         trainingPackage: { code: string; title: string } | null;
         unitGroups: Array<{ units: Array<{ code: string; title: string }> }>;
       }
       ```

    2. **apps/api/src/tga/dto/tga-unit.dto.ts**:
       ```typescript
       export interface TgaUnitDetail {
         code: string;
         title: string;
         status: string;
         supersededBy: string | null;
         elements: Array<{
           num: string;
           title: string;
           performanceCriteria: Array<{ num: string; text: string }>;
         }>;
       }

       export interface TgaUnitSearchResult {
         code: string;
         title: string;
         status: string;
       }
       ```

    3. **apps/api/src/tga/tga-api.client.ts**:
       - @Injectable() class `TgaApiClient`
       - Private `BASE_URL = 'https://training.gov.au/api'` (make configurable via ConfigService for testability)
       - Private `fetchWithRetry(url: string, retries = 3): Promise<any>` — implements exponential backoff (100ms, 200ms, 400ms) on 429 and 5xx responses; throws after all retries exhausted
       - Method `searchQualifications(query: string): Promise<TgaQualificationSearchResult[]>` — GET /qualifications?keywords={query}, returns array of results
       - Method `getQualificationDetail(code: string): Promise<TgaQualificationDetail>` — GET /qualifications/{code}
       - Method `getUnitDetail(code: string): Promise<TgaUnitDetail>` — GET /units/{code}
       - Method `searchUnits(query: string): Promise<TgaUnitSearchResult[]>` — GET /units?keywords={query}
       - All methods add `Accept: application/json` header
       - Log warnings on non-200 responses; never throw on 404 (return null for detail methods if 404)

    4. **apps/api/src/tga/tga.module.ts**:
       - @Module({ providers: [TgaApiClient], exports: [TgaApiClient] })
       - export class TgaModule {}

    5. **apps/api/src/app.module.ts** — add `TgaModule` to imports array
